# The League of Extraordinary Idiots - Points Adjuster

This extension automatically makes the proper scoring adjustments on the quick box matchup page on ESPN for members of the League of Extraordinary Idiots.

## Install

	$ npm install

## Development

    npm run dev chrome
    npm run dev firefox
    npm run dev opera
    npm run dev edge

## Build

    npm run build chrome
    npm run build firefox
    npm run build opera
    npm run build edge

## Environment

The build tool also defines a variable named `process.env.NODE_ENV` in your scripts.
